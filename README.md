# 📊 Healthcare Patient Analytics Dashboard

### 🔹 Project Overview
This project analyzes healthcare data to evaluate patient trends, hospital performance, and operational efficiency using data analytics techniques.

### 🔹 Problem Statement
Healthcare systems generate large volumes of patient data, but extracting meaningful insights is challenging. This project aims to analyze patient and operational data to identify patterns and improve healthcare performance.

### 🔹 Dataset
Healthcare dataset (Excel-based)<br>
Note: Dataset not uploaded due to size limitations.

### Tools & Technologies Used
* Microsoft Excel / CSV – Dataset storage and preprocessing
* Power BI – Power Query, Data modeling, DAX measures, and interactive dashboard3#s

### 🔹 Workflow
* Cleaned and transformed raw healthcare data.<br>
* Performed analysis on patient and operational metrics.<br>
* Evaluated relationships between variables.<br>
* Built dashboard to visualize insights.

### 🔹 Key Insights & Business Impact
* Analyzed patient data to identify trends in hospital performance and service utilization.<br>
* Identified relationships between length of stay and patient satisfaction, highlighting key factors affecting experience.<br>
* Evaluated hospital ratings and performance metrics, enabling comparison across facilities.<br>
